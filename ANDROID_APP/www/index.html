Guida all'Uso

AI NutriTracker è un'applicazione semplice, snella e senza fronzoli, progettata per offrirti solo quello che serve realmente per tracciare la tua alimentazione e forma fisica senza complicazioni. È organizzata in 3 sezioni principali:

Sezione Dati (Oggi)

Inserisci cosa hai mangiato nel box di testo libero (es. "pasta al pesto e un uovo"). Puoi anche usare la dettatura vocale col microfono! L'Intelligenza Artificiale farà il calcolo esatto. Se preferisci, c'è un inserimento manuale.

Sezione Trend

Mostra l'andamento del peso, delle calorie assunte e della ripartizione dei macronutrienti negli ultimi 7 giorni, nelle ultime 8 settimane e il registro mensile. Clicca su "Chiedi parere ad AI" per ricevere un'analisi dettagliata sui tuoi progressi e consigli strategici personalizzati.

Sezione Registro

Contiene l'archivio storico di tutti i pasti inseriti. Puoi filtrare per testo, visualizzare il log dettagliato scompattato dall'AI, modificare o eliminare qualsiasi pasto.

I punti di forza e le caratteristiche chiave di AI NutriTracker:

  • Decomposizione Automatica dei Piatti

    Non serve cercare e pesare ogni singolo ingrediente. Scrivi o detta un pasto intero e l'AI lo scompone stimando porzioni standard SINU/LARN regolate sul tuo profilo fisico.

  • Coerenza Matematica Rigorosa

    A differenza delle app tradizionali dove i macro inseriti dagli utenti non corrispondono mai alle calorie, la nostra AI garantisce la coerenza termodinamica perfetta: (Proteine * 4) + (Carboidrati * 4) + (Grassi * 9) = Calorie totali.

  • Privacy e Proprietà dei Dati

    Nessun server dell'app. Le tue chiavi API e impostazioni risiedono solo su localStorage, e l'intero diario alimentare è memorizzato sul tuo dispositivo tramite IndexedDB. I dati dei pasti vengono inviati direttamente ai server Google tramite la tua API Key personale per l'analisi, senza che l'autore dell'app o terze parti possano accedervi.

Per abilitare l'analisi dei pasti tramite Intelligenza Artificiale, è necessario inserire una chiave Gemini API Key personale. È gratuita, non richiede alcuna carta di credito o metodo di pagamento, e garantisce:

Longevità e Indipendenza

1 Accedi a Google AI Studio

Visita il portale per gli sviluppatori di Google: aistudio.google.com ed effettua il login con il tuo account Google.

2 Crea la chiave API Key

Clicca sul pulsante "Get API Key" o "Create API Key". Se richiesto, crea un nuovo progetto e copia la chiave generata (il codice inizierà con AIzaSy...).

3 Inserisci la chiave nell'app

Torna in questa applicazione, apri le Impostazioni (icona ingranaggio in alto), espandi la sezione "Gemini API Key", incolla il codice nel campo ed è fatta!

Archiviazione dei Dati

Tutti i pasti, le misurazioni del peso e i target fisici vengono salvati in locale nel database IndexedDB e localStorage del browser. Se svuoti la cache o i dati del browser per questo sito, i tuoi dati inseriti potrebbero andare persi. Puoi effettuare backup periodici esportando il database in formato JSON dalle Impostazioni.
